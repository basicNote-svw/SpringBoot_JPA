# Spring Boot

서버만들 수 있는 언어와 프레임워크
---
- 빠른 비동기처리 서버를 만들고 싶으면 Node.js

- AI 서비스에 붙일 서버가 필요하면 Python

- 리액트가 좋으면 Next.js

- 성능과 동시성 좋아하면 Erlang류, Rust, Go

<br>

<br
 
Java 와 Spring Boot
---
- Java 언어 : 코드 정리와 재사용을 강요하는 역사깊은 언어

- Spring Boot : 서버 만드는 프레임워크

  - 코드를 짠다기보다 배치만 하면 돌아감

    - 기술처럼 외워쓸 수 있다는 점이 장점이자 단점

쇼핑몰 프로젝트
---
- 웹에 있는 거의 모든 기능은 90% 확률로 게시판 기능과 똑같음

  - 당근마켓도 그냥 물건사진 올리는 게시판

  - 인스타그램도 친구게시물 보여주는 게시판

  - 쇼핑몰도 상품사진 올리는 게시판

  - 쇼핑몰에서 주문버튼 누르는 것도 게시판 글발행하는것과 같음
